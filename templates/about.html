ARTHAMUDRA

Malayalam Sign Language Translator

ARTHAMUDRA is an advanced Malayalam Sign Language (MSL) translation platform designed to bridge the communication gap between the hearing-impaired community and the wider society. Using cutting-edge deep learning models, ARTHAMUDRA translates Malayalam Sign Language gestures into text, enabling seamless interaction and accessibility.

Features

  • Real-time sign language recognition
  • Accurate translation of Malayalam Sign Language
  • Easy-to-use web-based interface
  • Flask-powered backend for seamless processing

Our Mission

We aim to create an inclusive society where individuals with hearing impairments can communicate effortlessly using Malayalam Sign Language. By leveraging AI and machine learning, SPARSHA empowers users with real-time translation and enhances accessibility.
